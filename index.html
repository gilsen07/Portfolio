Football Prediction Model

Developed a machine learning model to predict football match outcomes using historical match data, feature engineering, and model evaluation techniques. Implemented data preprocessing, model validation, and performance tuning to achieve reliable predictions. Includes a simple front-end for user inputs and predictions.
Python scikit-learn pandas XGBoost Machine Learning

Overview

Reads match data (e.g., final_dataset.csv), predicts home-win (target: FTR == 'H') using a pipeline that includes data cleaning, scaling numeric features (HTGD, ATGD, HTP, ATP, DiffLP), and encoding recent-match outcomes (HM1..HM3, AM1..AM3).

Highlights

  • Models: LogisticRegression, SVC, XGBoost (tuned with GridSearchCV).
  • Evaluation: F1-score (positive = 'H'), accuracy.
  • Persistence: Trained model is saved for later inference (e.g., model.pkl).
  • Front-end: Simple UI accepts inputs and returns predictions from the trained model.
  • Developer notes: ensure tuned estimator is persisted and validate accuracy computation during tests.
